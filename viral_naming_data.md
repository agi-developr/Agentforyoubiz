# Viral Naming Dataset: The "Golden Goose" Engine

This dataset is designed to be fed into a creative process (or LLM) to generate high-velocity, sticky brand names.

## 1. High-Velocity Linguistics (Gen Z / Alpha Slang)
*Words that imply speed, impact, or status transformation.*

*   **Verbs of Motion:** Yeet, Send, Dip, Skrt, Zoom, Slide, Pull up, Ghost, Dash, Bolt, Bounce.
*   **Status Indicators:** Aura, Riz, Mid, Peak, Prime, Goated, Elite, Valid, Locked.
*   **Emotional States:** Manic, Chill, Numb, feral, Unhinged, Cozy, Pressed, Shook, Woke.
*   **Social Dynamics:** Opp, Fam, Gang, Squad, Cult, Hive, Stan, NPC, Main.

## 2. Gaming & Digital Culture Metaphors
*Travel is increasingly gamified. These terms bridge the gap between "location" and "level."*

*   **Structure:** Server, Lobby, Map, Biome, Zone, Realm, Level, Stage.
*   **Action:** Spawn, Respawn, Loot, Grind, buff, Nerf, Lag, Glitch, Hack, Cheat.
*   **Identity:** Avatar, Skin, Build, Loadout, Stat, Rank, Tier, XP, HP, Alt.
*   **Narrative:** Quest, Side-quest, Lore, Canon, Arc, Plot, Cutscene, Dialog.

## 3. The "Escapist" Psychographic Dictionary
*Words that trigger the desire to leave the current reality.*

*   **Disappearance:** Void, Abyss, Static, Blank, Null, Zero, Gone, Away, Out, Off.
*   **Transformation:** Shift, Morph, Flux, Warp, Bend, Fold, Turn, Snap.
*   **Alternate Realities:** Ether, Astral, Dream, Lucid, Liminal, Beyond, Other, Else.
*   **Sanctuary:** Haven, Base, Nest, Pod, Cell, Hub, Node, Core.

## 4. Phonesthetic "Sticky" Sounds
*Sounds that are statistically more likely to be remembered or go viral.*

*   **Plosives (Explosive start):** P, K, T, B (e.g., Pop, Kick, Tick, Bam).
*   **Sibilants (Smooth/Fast):** S, Z, Sh (e.g., Zoom, Swift, Shift).
*   **Short Vowels (Punchy):** A (as in cat), I (as in bit), U (as in cut).
*   **Double Letters:** ZZ, BB, MM (e.g., Buzz, Bubble).

## 5. Proven Startup Naming Patterns (For Reconstruction)
*   **The "Drop Vowel":** Tmblr, Flckr, Grindr -> Trvl, Lvls, Wrld.
*   **The "Personification":** Oscar, Casper, Sid -> "Alfred", "Marco", "Joy".
*   **The "Command":** Go, Get, Do -> "GoPuff", "GetAway".
*   **The "Compound":** Snap-chat, Face-book -> "Trip-Tok"? (Avoid generic compounds).
*   **The "Suffix":** -ify, -ly, -io, -ia, -is.

## 6. Curated Combination Prompts
*Use these formulas to mix the data above:*

*   **Formula A (The "Glitch"):** [Digital Metaphor] + [Motion Verb]
    *   *Examples:* GlitchRun, ServerHop, LagSwitch.
*   **Formula B (The "Narrative"):** [Identity] + [Structure]
    *   *Examples:* ArcMap, LoreZone, PlotPoint.
*   **Formula C (The "Raw State"):** [Emotional State] + [Sanctuary]
    *   *Examples:* FeralBase, ManicHub, ChillPod.
